📝 Rekenmachine

This project is a simple web application built using HTML, CSS, and JavaScript.
It provides an interactive user experience with a clean and responsive design.

🚀 Features
Interactive user interface
Responsive design (mobile-friendly)
Dynamic content with JavaScript
Easy to use and lightweight

⚙️ Installation

Clone the repository:

git clone https://github.com/Omer-Girginer/Rekenmachine

Open the project folder:

cd Rekenmachine

Run the project:
Just open the index.html file in your browser.

🎯 Purpose

This project was created to practice fundamental web development skills such as structuring with HTML, styling with CSS, and adding interactivity using JavaScript.
